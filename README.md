# InterviewFlow AI 🚀

An **AI-powered interview platform** that enables organizations to conduct intelligent candidate screening with **AI interviews, coding assessments, and real-time emotion detection**.

The platform provides tools for both **candidates and interviewers**, enabling automated hiring workflows and better interview insights.

---

# ✨ Features

## 👨‍💻 Candidate Features

* **AI Mock Coding Interview**

  * Solve coding problems in a built-in live code editor
  * AI evaluates correctness and complexity

* **Tech Interview Preparation**

  * Upload resume
  * AI generates interview questions based on skills

* **Resume → Job Matching**

  * Upload resume
  * System finds relevant jobs automatically

* **Candidate Dashboard**

  * Track interviews
  * Join interview sessions
  * View interview history

---

## 🧑‍💼 Interviewer Features

* **Schedule Interviews**
* **Invite Candidates**
* **Manage Interview Sessions**
* **Review Candidate Performance**

---

## 🤖 AI Interview System

Powered by **Vapi AI speech-to-speech agents**

Features:

* AI asks technical questions

# 🚀 How to Run Locally

### 1️⃣ Clone repository

```
git clone https://github.com/yourusername/interviewflow-ai.git
cd interviewflow-ai
```

---

### 2️⃣ Install frontend

```
cd frontend
npm install
npm run dev
```

---

### 3️⃣ Start backend

```
cd backend
npm install
npm start
```

---

### 4️⃣ Start emotion detection API

```
cd emotion-service
python emotion_api.py
```
